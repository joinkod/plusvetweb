# Plus Vet — Sitio Web Oficial

Sitio web institucional de **Plus Vet Clínica Veterinaria**, especializada en perros y gatos en Santa Rosa de Cabal, Risaralda, Colombia.

## Stack

- HTML5 + CSS3 + JavaScript vanilla
- Sin frameworks ni dependencias de build
- Fonts: Poppins vía Google Fonts
- Íconos: Font Awesome 6.5

## Estructura

```
plusvet/
├── index.html
├── vercel.json
├── .gitignore
├── README.md
├── PLUSVET_PROJECT.md   ← Documentación completa del proyecto
└── assets/              ← Logos SVG y recursos estáticos
```

## Desarrollo local

No requiere servidor de build. Abrir `index.html` directamente en el navegador, o usar una extensión como **Live Server** en VS Code.

## Despliegue

El proyecto está configurado para despliegue automático en **Vercel**. Cada push a `main` genera un nuevo deploy.
